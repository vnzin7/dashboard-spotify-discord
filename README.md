# Dashboard Spotify + Discord

Dashboard pessoal integrado com Spotify e Discord, mostrando em tempo real:
- Música atual no Spotify
- Top músicas e artistas
- Playlists favoritas
- Perfil do Discord
- Estatísticas do servidor Discord
- Membros online

## Tecnologias
- Next.js 14
- TypeScript
- Tailwind CSS
- Spotify API
- Discord API

## Deploy
Deploy automático na Vercel conectado ao GitHub.

## Configuração
1. Clone o repositório
2. Copie `.env.local.example` para `.env.local`
3. Preencha as credenciais
4. Execute `npm install`
5. Execute `npm run dev`
