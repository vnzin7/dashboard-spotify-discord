# Discord Token Manager

Painel web para gerenciar tokens do Discord com verificação de status em tempo real.

## Funcionalidades

✅ Adicionar múltiplos tokens via texto
✅ Verificar status (online/offline/inválido)
✅ Visualizar informações do usuário
✅ Exportar/Importar tokens em JSON
✅ Webhook Discord para notificações
✅ Auto-verificação a cada 30 segundos

## Acesso

Site hospedado na Vercel
